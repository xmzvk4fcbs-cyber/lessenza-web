🗓️ Google Kalendar i Email ▸
Učitavanje…

Kad povežeš Google nalog, termini idu u tvoj kalendar, a klijenti dobijaju potvrdu na email — sve kroz tvoj Gmail.

Prvi put? Kako podesiti
  1. Otvori Google Cloud Console i napravi novi projekat "Lessenza".
  2. U istom projektu: APIs & Services → Library → uključi Calendar API i Gmail API.
  3. OAuth consent screen: External + tvoj email kao test user.
  4. Credentials → Create OAuth client → Web application. Authorized redirect URI: https://lessenza.me/api/admin/google-callback.
  5. Copy Client ID i Client Secret.
  6. U Netlify → Site configuration → Environment variables → dodaj GOOGLE_OAUTH_CLIENT_ID i GOOGLE_OAUTH_CLIENT_SECRET.
  7. Redeploy → vrati se ovdje → "Poveži Google".
